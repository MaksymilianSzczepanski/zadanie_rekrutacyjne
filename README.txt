Dzień dobry,

Projekt został napisany w framework Laravel. W celu uruchamiania projektu wymagane jest posiadanie composera. 
Następnie należy zaktualizować i pobrać zależności poleceniem composer update. 
Żeby uruchomić server należy wpisać w głównym katalogu w wierszu polecie php artisan serv. Wgrywane pliki znajdują się w folderze projektu storage\files.
